¿Cómo colocar videos en HTML?

En HTML, para agregar videos usamos la etiqueta < video >

Para que el video funcione, necesitamos indicar su ubicación usando el atributo src, que define el archivo de video.
Además, podemos agregar otros atributos como controls para mostrar controles de reproducción.

Pasos para colocar un video:

1. Usa la etiqueta < video >.

2. Usa el atributo src para especificar la ubicación del archivo de video.

3. Agrega el atributo controls para permitir que el usuario controle la reproducción (pausar, reproducir, cambiar volumen, etc.).

Ejemplo de código:

< video src="URL_del_video" controls> < / video >
