📋 Listas en HTML

Las listas en HTML son una herramienta fundamental para organizar y presentar información de manera estructurada, ya sea de forma ordenada o desordenada.

¿Qué son las Listas en HTML?

En HTML, las listas se utilizan para agrupar elementos de manera secuencial o no secuencial. Existen dos tipos principales de listas: ordenadas y desordenadas. Las listas ordenadas presentan los elementos con numeración, mientras que las listas desordenadas usan viñetas para presentarlos.

Ejemplos de Listas

A continuación te mostramos cómo se crean las listas ordenadas y desordenadas en HTML.

Lista Ordenada:

Las listas ordenadas usan el elemento <ol> y numeran automáticamente los elementos dentro de ella.

        
          <ol>
            <li>Inicio</li>
            <li>Contenido</li>
            <li>Resumen</li>
          </ol>

Resultado de la lista ordenada:

  1. Inicio
  2. Contenido
  3. Resumen

Lista Desordenada:

Las listas desordenadas usan el elemento <ul> y presentan los elementos con viñetas.

        
          <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
          </ul>

Resultado de la lista desordenada:

  • HTML
  • CSS
  • JavaScript

Pasos para Crear Listas en HTML

  1. Crear una lista ordenada: Utiliza el elemento <ol> para crear una lista numerada. Dentro de ella, coloca cada ítem con el elemento <li>.
  2. Crear una lista desordenada: Utiliza el elemento <ul> para una lista con viñetas. Al igual que en las listas ordenadas, cada ítem se coloca dentro de un elemento <li>.
  3. Personalizar el estilo: Puedes cambiar el estilo de las viñetas en listas desordenadas, y también modificar el formato de los números en las listas ordenadas mediante CSS.

Buenas Prácticas para las Listas

  • Usa listas para agrupar elementos relacionados: Asegúrate de que los elementos de la lista sean coherentes entre sí y tengan un propósito común.
  • Evita listas demasiado largas: Si tienes muchos elementos, considera dividir las listas en subsecciones o usar otros elementos de organización como tablas o tarjetas.
  • Estiliza las listas: Puedes personalizar el estilo de las listas para hacerlas más atractivas, como cambiar las viñetas o los números, o añadir espaciado entre los ítems.
