¿Cómo colocar imágenes en HTML?

En HTML, para colocar una imagen usamos la etiqueta < img >.

El atributo principal que necesita la etiqueta < img > es src, que indica la ubicación de la imagen. También se recomienda usar el atributo alt para describir la imagen.

Pasos para colocar una imagen:

  1. Usa la etiqueta < img >.

  2. Define el atributo src con la URL o la ruta del archivo de la imagen.

  3. Agrega el atributo alt para describir la imagen, lo cual es útil para accesibilidad y cuando la imagen no se carga.

Ejemplo de código:

< img src="colocar una url para la imagen" alt="Descripción de la imagen" >
