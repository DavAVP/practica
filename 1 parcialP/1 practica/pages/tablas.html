¿Qué es una tabla en html?

Las tablas son un recurso muy útil para organizar datos tabulados y definir la disposición del texto e imágenes
dentro de una página HTML. Están formadas por una o más filas, y cada fila se divide en una o varias celdas.

Una tabla facilita la localización rápida y sencilla de valores relacionados entre distintos tipos de información.
Por ejemplo, puede mostrar la relación entre una persona y su edad, o entre los días de la semana y el horario
de una piscina municipal.

¿Como se programa en html una tabla?

para comenzar debemos seguir los siguientes pasos:

  • El contenido de una tabla se encierra entre las etiquetas < table > y < / tabla >,
    estas deben colocarse dentro de la sección de tu código HTML.

  • La unidad más pequeña dentro de una tabla es la celda, la cual se define mediante la
    etiqueta < td >, que proviene de "table data" (dato de tabla). Debes colocar estas etiquetas
    dentro de las etiquetas < table > para agregar contenido a tu tabla.

    • < table >
    < th > Nombre < / th >
    < th > Apeillo < / th >
    < th > Edad < / th >
    < / table >

¿Qué significan las etiquetas <tr> y <th>?

  • La etiqueta < tr > se utiliza para definir una fila dentro de una tabla.
    Cada fila puede contener una o más celdas, ya sean de encabezado o de datos.

  • La etiqueta < th > se usa para crear una celda de encabezado en una tabla.
    Estas celdas normalmente se muestran en negrita y centradas de forma predeterminada.
    Sirven para describir el contenido de las columnas o filas.

    • < table >
    < tr >
    < th > Nombre < /th >
    < th > Apellido < /th >
    < th > Edad < /th >
    < /tr >
    < tr >
    < td > Laura < /td >
    < td > Gómez < /td >
    < td > 30 < /td >
    < /tr >
    < /table >
