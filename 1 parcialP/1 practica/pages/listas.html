¿Qué son las listas en HTML?

En HTML, las listas son una manera de organizar elementos de manera estructurada. Se pueden crear dos tipos de listas: listas ordenadas y listas desordenadas.

Listas ordenadas

Las listas ordenadas se usan cuando el orden de los elementos es importante. Cada elemento se presenta con un número o una letra, según el tipo de lista ordenada.

Para crear una lista ordenada en HTML, se usa la etiqueta < ol > < / ol> (ordered list), y cada elemento de la lista se define con la etiqueta < li > < / li > (list item).

Ejemplo de una lista ordenada:

< ol >
< li > primer paso < / li >
< li > segundo paso < / li >
< li > tercer paso < / li >
< /ol >

Este es un ejemplo de una lista ordenada que enumera pasos en un proceso:

  1. Primer paso
  2. Segundo paso
  3. Tercer paso

Listas desordenadas

Las listas desordenadas se usan cuando el orden de los elementos no es importante. En lugar de números, los elementos se muestran con viñetas (puntos).

Para crear una lista desordenada en HTML, se usa la etiqueta < ul > < / ul > (unordered list), y cada elemento de la lista se define con la etiqueta < li > < / li >.

Ejemplo de una lista desordenada:

< ul >
< li > Manzanas < / li >
< li > Plátanos < / li >
< li > Cerezas < / li >
< / ul >

Este es un ejemplo de una lista desordenada con frutas:

  • Manzanas
  • Plátanos
  • Cerezas

Con eso termina la guia de listas ordenadas y desordenadas.
