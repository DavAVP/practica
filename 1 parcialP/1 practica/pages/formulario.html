¿Qué es un formulario en HTML?

Un formulario en HTML permite a los usuarios introducir información en una página web.
Se usa para enviar datos como nombres, correos electrónicos o números de teléfono a un servidor o base de datos.
Son muy comunes en sitios web donde se necesita recolectar información del visitante.

¿Cómo se crea un formulario?

Para crear un formulario en HTML, se deben seguir estos pasos básicos:

  • Se comienza con la etiqueta < form > que encierra todo el formulario.

  • Para cada campo se usa una etiqueta < label > que indica qué debe escribir el usuario.

  • Luego se utiliza la etiqueta < input > para que el usuario pueda escribir su información.

  • El botón de envío se crea con un < input > de tipo "submit".
